Project Brainstorm

Theodore
  • Named after the protagonist from the 2013 film Her (dir. Spike Jonze), this machine will write letters for you in your handwriting. In order to achieve this, it will need to house and intake paper slips where there can be standardized collection of all the letters in your handwriting--capital, lowercase. Using a camera and optical character recognition/background subtraction, the device will be able to recombine and reproduce letters in that handwriting using some writing implement controlled by motors. I also want to integrate natural-seeming variations in letters--say, +/- 5% variation in width and/or height. Theodore thus can create a letter-profile or custom font for users. Subsequently, through a simple web application, a user can log in and type their message for Theodore to transcribe. This type of device is already used by businesses for professional and consumer applications around the world, including companies like Zappos, Snowflake, and Sotheby's to great success. However, the machines cost thousands of dollars, so I am hoping to create a more accessible version that maintains the same core functionalities.
